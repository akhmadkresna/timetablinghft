
	PROGRAM ARGUMENTS
	-------------------------------------
	
	1)	File name of the problem instance INCLUDING DIRECTORY NAME						mandatory
	
			use 'ALL' if you want to batch-execute all problem instances
			in the folder 'instances'!
	
	
	2)	Number of iterations to perform													optional
	
			defaults to 200
			
			
	3)	Number specifying how often the program is batch-executed						optional
	
			defaults to 1
	
			
	4)	Sleep time between iterations to be able to better follow console output		optional
	
			defaults to 0
	
	
	5)	Name of the directory to read initial solutions from							optional
	
			feature will not be used if not specified
