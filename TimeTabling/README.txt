
	PROGRAM ARGUMENTS
	-------------------------------------
	
	1)	File name of the problem instance contained in the folder "instances"			mandatory
			(use 'ALL' if you want to batch-execute all problem instances!)
	
	2)	Number of iterations to perform													optional
			(defaults to 1000)
			
	3)	Sleep time between iterations to be able to better follow console output		optional
			(defaults to 0)
	
	4)	Name of the directory to read initial solutions from							optional
			(feature will not be used if not specified)
